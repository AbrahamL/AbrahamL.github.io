Sun 25 January 2015
By Abraham

快速开始

我们推荐你通读本文档，下面是为了那些急切的人提供的快速入门教程。

安装

用你喜欢的终端在Python2.7.x或Python 3.3+上安装Pelican（如果需要可以安装Markdown），命令为：

sudo pip install pelican markdown

创建项目

首先为你的项目选一个名字，在本地创建一个合适的目录，并进入该目录：

mkdir -p ~/projects/yoursite
cd ~/projects/yoursite

通过pelican-quickstart命令创建一个项目框架，该命令会问你几个关于你的网站的问题：

pelican-quickstart

对中括号中有默认值的问题，你可以按回车键接受那些默认值。在关于网站前缀的问题，输入你预想的域名(例如： http://example.com)。

创建文章

当创建一些内容后，你才可以运行Peliacn。用你喜欢的文本编辑器创建你的第一篇文章，内容如下：

Title: My First Review
Date: 2015-01-24
Category: review

Following is a review of my favorite mechanical keyboard.

上面给的内容是Markdown格式的，将它保存为～/projects/yoursite/content/keyboard-review.md

生成你的站点

在你的项目文件夹，运行pelican命令来生成你的站点：

pelican content

你的网站内容已经在output文件夹内生成了。（你可能会看到一些关于feed的警告，但这在本地开发时是正常的，现在可以忽略它们。）

预览你的网站

新打开一个终端，进入output文件夹后输入下列命令来运行内建的web服务器：

cd ~/projects/yoursite/output
python -m SimpleHTTPServer

用你的浏览器访问http://localhost:8000/来预览你的网站。

继续阅读剩余的文档来获取更多的细节，并且查看Pelican wiki's Tutorials 页面来获取社区出版的教程。
