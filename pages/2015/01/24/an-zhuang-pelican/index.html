安装Pelican

Sat 24 January 2015
By Abraham

Pelican目前最适合在Python2.7.x上运行，不支持之前版本的Python。

你可以通过多种方式安装Pelican，其中最简单的是通过pip安装

sudo pip install pelican

尽管上面的方法是最简单的，但是我们推荐在安装Pelican之前通过virtualenv为其创建一个虚拟环境。在你安装了virtualenv之后，在终端中输入下列命令为Pelican创建一个新的虚拟环境：

virtualenv ~/virtualenvs/pelican
cd ~/virtualenvs/pelican
source bin/activate

在虚拟环境创建并激活后，Pelican就可以通过上面的方法（pip install pelican）安装。或者通过源代码来安装：

cd path-to-Pelican-source
Python setup.py install

如果你安装了Git并且希望安装最新的Pelican而不是稳定版的，执行下列命令：

pip install -e "git+https://github.com/getpelican/pelican.git#egg=pelican"

Pelican安装成功后，你可以运行pelican --help来查看基本的用法选项。要获取更多的细节，可以查看Publish部分。

可选包

如果你希望使用Markdown语法编辑文档，你需要安装Markdown库：

pip install Markdown

安装 Typogrify库后，可以在你的配置文件中激活排版增强。

依赖

在安装Pelican后，下列包应当已经自动地安装完成了：

  • feedgenerator,用于生成Atom feeds
  • [jinja2],用于模板支持
  • pygments,用于语法高亮
